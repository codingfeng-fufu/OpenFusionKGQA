### 问题2: 数据模型 Fixtures 参数不匹配 ✅

**问题描述**:
- Entity: 使用 `name` 而不是 `title`
- 所有模型: 缺少 `short_id` 参数
- Community: 使用 `entities` 而不是 `entity_ids`
- Community: 缺少 `level`, `parent`, `children` 参数

**修复方案**:
```python
# 修复前
Entity(id="e1", name="GraphRAG", type="技术")

# 修复后
Entity(id="e1", short_id="e1", title="GraphRAG", type="技术")
```

**修复的 Fixtures**:
- ✅ `sample_entities` - 使用 `title` 和 `short_id`
- ✅ `sample_relationships` - 添加 `short_id`
